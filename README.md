# OpenClaw Skill: task-tracker

> This is an agent skill for [OpenClaw](https://openclaw.ai).

Task tracker workflow for implementing backend APIs from tickets (Redmine, Jira, Linear, etc.).

## Installation

```bash
clawhub install guoway/openclaw-skill-task-tracker
```

## Features

- Requirement clarification (interview-style)
- Feature branch creation per ticket
- Implementation workflow
- Testing + coverage report
- Commit & Push
- Update ticket status
